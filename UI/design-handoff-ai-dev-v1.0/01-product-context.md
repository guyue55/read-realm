# 01 产品上下文

## 产品定位

个人小说聚合阅读平台，面向长期小说/网文/长篇阅读场景，支持：

- 本地 TXT / EPUB 上传
- 粘贴链接解析
- 免费搜索发现与候选入口
- 章节识别、净化、质量提示和手动修正
- Web / 手机浏览器 / PWA 跨屏沉浸式阅读
- 离线缓存、阅读进度恢复、书签笔记
- AI 阅读增强：章节摘要、前情提要、人物关系、术语解释

## MVP 第一目标

MVP 不追求完整内容生态，先验证三件事：

| 目标     | 验证内容                               |
| -------- | -------------------------------------- |
| 能导入   | TXT / EPUB 能上传，URL 能尝试解析      |
| 能解析   | 能识别目录、章节、正文，并允许手动修正 |
| 能舒服读 | 阅读排版稳定、可续读、可离线、可调主题 |

## 首发端范围

第一阶段是 **Web/PWA 跨屏 MVP**，不是单纯桌面 Web：

- 桌面浏览器：增强布局、三栏阅读、拖拽上传、快捷键
- 手机浏览器：单手阅读、底部菜单、点击区域、安全区、弱网可读
- PWA：可安装、离线打开、缓存章节可读

## 第一条主链路

```text
打开应用
  -> 书架
  -> 导入 TXT / EPUB / URL
  -> 解析预览
  -> 手动修正章节
  -> 加入书架
  -> 阅读页
  -> 自动保存进度
  -> 下次继续阅读
```
