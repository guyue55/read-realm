# 小说阅读平台 UI 真矢量 SVG 套图 v0.2

本包是逐元素可编辑的矢量复刻版。

## 特点

- 不再使用 `<image>` 标签
- 不内嵌 PNG / JPG / base64
- 所有界面由 SVG 原生元素组成：`rect`、`text`、`path`、`circle`、`line`、`g`
- 每个页面分组命名，适合在 Figma、Illustrator、Inkscape、浏览器开发者工具中继续编辑
- 文本保持为 SVG `<text>`，颜色、圆角、线条、卡片、按钮均可单独选择修改

## 说明

这是结构化矢量复刻，不是图片自动描摹。目标是让 UI 元素可维护、可继续设计和前端落地，而不是逐像素复制噪点、纹理或截图压缩痕迹。
