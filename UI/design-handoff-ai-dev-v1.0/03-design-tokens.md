# 03 Design Tokens

开发必须使用 tokens，不允许 AI 自由发挥颜色、圆角、字号、正文宽度、底部安全区和阅读层级。

## 颜色

| Token           |        值 | 用途             |
| --------------- | --------: | ---------------- |
| `paper`         | `#F8F8F5` | 纸白背景         |
| `sepia`         | `#F4ECD8` | 默认米黄阅读背景 |
| `green`         | `#DDEBD6` | 护眼绿           |
| `warmGray`      | `#E8E3DA` | 暖灰             |
| `dark`          | `#1E1E1E` | 默认夜间背景     |
| `textPrimary`   | `#2F2A24` | 正文主文本       |
| `textSecondary` | `#6F665B` | 弱文本           |
| `accent`        | `#9A6A3A` | 主强调色         |
| `accentDark`    | `#D2A66A` | 夜间强调色       |

## 字体排版

| 场景     | 默认字号 | 行高 |  段距 |
| -------- | -------: | ---: | ----: |
| 手机     |     18px | 1.65 | 0.5em |
| 平板     |     19px |  1.6 | 0.5em |
| 桌面     |     20px |  1.7 | 0.5em |
| 大字模式 |     24px | 1.75 | 0.6em |

字号必须支持步进或滑杆，不允许只做“小 / 中 / 大”。

## 正文宽度

| 场景     | 规则                                                      |
| -------- | --------------------------------------------------------- |
| 手机竖屏 | 左右 18–24px 边距                                         |
| 手机横屏 | 最大正文宽度 720px，居中                                  |
| 平板     | 最大正文宽度 760px                                        |
| 桌面单栏 | 680–820px                                                 |
| 桌面三栏 | 固定 1280px 工作区；目录 240px / 正文 700px / AI 约 338px |
