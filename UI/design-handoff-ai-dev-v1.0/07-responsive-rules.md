# 07 响应式规则

## 断点模型

```ts
type ReaderViewport = "mobile" | "tablet" | "desktop" | "wide";
```

| Viewport |      宽度建议 | 阅读布局                         |
| -------- | ------------: | -------------------------------- |
| mobile   |     `< 768px` | 单栏、底部菜单、点击区、半屏设置 |
| tablet   |  `768–1023px` | 单栏/双栏可选、抽屉目录          |
| desktop  | `1024–1439px` | 单栏居中、可展开目录与 AI        |
| wide     |   `>= 1440px` | 1280px 固定工作区、三栏布局      |

## 移动端硬规则

- 底部交互区域必须加 `padding-bottom: env(safe-area-inset-bottom)`。
- 不允许固定截图尺寸死写。
- 不允许把桌面三栏强压到手机宽度。
- 功能页可使用内部滚动容器，阅读页保持正文优先。

## 桌面端硬规则

- 三栏阅读工作区固定 1280px，不因窗口变窄压扁。
- 窄屏下允许横向滚动或降级为单栏，不允许隐藏右栏后仍占位挤压正文。
- 正文最大宽度 760–820px。
- 宽屏不允许正文铺满 1440px。
