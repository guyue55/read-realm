# FTS5 Search Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement full-text search for books using SQLite FTS5.

**Architecture:** 
- Use `SearchRepository` to perform raw SQL `MATCH` queries on the `books_search_v` virtual table.
- Retrieve full book details from the main `books` table using IDs returned by the search.
- Expose search via `GET /search?q=...` in `SearchController`.

**Tech Stack:** NestJS, LibSQL (SQLite), Drizzle ORM.

### Task 1: Implement SearchRepository

**Files:**
- Create: `apps/api/src/modules/search/search.repository.ts`

- [ ] **Step 1: Write SearchRepository implementation**

```typescript
import { Injectable, Inject } from '@nestjs/common';
import { DRIZZLE } from '../database/database.module';
import { LibSQLDatabase } from 'drizzle-orm/libsql';
import * as schema from '../database/schema';
import { inArray } from 'drizzle-orm';

@Injectable()
export class SearchRepository {
  constructor(@Inject(DRIZZLE) private db: LibSQLDatabase<typeof schema>) {}

  async searchBooks(query: string) {
    if (!query) return [];

    // FTS5 search
    const results = await this.db.execute({
      sql: 'SELECT id FROM books_search_v WHERE books_search_v MATCH ? ORDER BY rank',
      args: [query],
    });

    const ids = results.rows.map((row) => row.id as string);

    if (ids.length === 0) return [];

    // Fetch full book details
    return this.db.query.books.findMany({
      where: inArray(schema.books.id, ids),
    });
  }
}
```

### Task 2: Implement SearchController

**Files:**
- Create: `apps/api/src/modules/search/search.controller.ts`

- [ ] **Step 1: Write SearchController implementation**

```typescript
import { Controller, Get, Query } from '@nestjs/common';
import { SearchRepository } from './search.repository';

@Controller('search')
export class SearchController {
  constructor(private readonly searchRepository: SearchRepository) {}

  @Get()
  async search(@Query('q') q: string) {
    return this.searchRepository.searchBooks(q);
  }
}
```
