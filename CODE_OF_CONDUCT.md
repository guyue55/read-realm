# 贡献者行为准则 (Contributor Covenant Code of Conduct)

## 我们的承诺

为了营造一个开放、友好、包容和健康的社区环境，作为本项目的维护者和贡献者，我们承诺：不论任何人的背景、性别、性别认同、性取向、残疾状态、外貌、体型、种族、民族、年龄、宗教或国籍，我们在社区中的交流合作都将免受骚扰。

我们致力于让所有参与者都拥有一段积极、建设性的体验。

---

## 我们的标准

有助于为社区营造积极环境的行为包括：

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 礼貌地接受建设性的批评
- 关注对整个社区最有利的事情
- 对其他社区成员展现同理心

不可接受的行为包括：

- 使用性暗示的语言或图像，以及不受欢迎的性关注或性骚扰
- 恶意贬低、侮辱、挑衅、人身攻击以及政治性或个人针对性言论
- 公开或私下的骚扰
- 未经明确许可，发布他人的私人信息（如物理地址或电子邮箱地址）
- 其他在专业和社区环境中合理被视为不当的行为

---

## 我们的职责

项目维护者有责任澄清和执行我们可接受行为的标准。对于任何他们认为不当、具有威胁性、冒犯性或有害的行为，他们将采取公平和公正的纠正措施。

项目维护者有权且有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并保留暂时或永久终止任何违规者参与社区的权利。

---

## 适用范围

本行为准则适用于项目的所有公共空间，同时也适用于代表项目或其社区进行交流的私人空间。例如，使用官方的社交媒体账号、电子邮件，或者在线下活动中作为项目的代表。

---

## 纠正与执行

如果发现、遇到任何违反本行为准则的行为，请通过邮件私下联系项目维护者：`guyue_admin@example.com`（请根据实际维护者邮箱替换）。所有投诉都将被及时审查和调查，并以保密和尊重隐私的方式处理。

项目维护者将真诚地履行职责，维护我们社区的和谐与健康发展。
